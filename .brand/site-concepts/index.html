KokoChat 下载页 · 三个概念设计

点开任意一个看完整页面。当前线上版本是更克制的中间路线。

Option A
Playful · 玩具感

大鸭子 + sticker 飘动装饰 + 圆胖大字 + neo-brutalism 黑边粗阴影按钮。最强品牌温度,适合社区/二次元向。

Option B
Terminal · 极客感

黑底 + 等宽字 + 伪命令行 hero + 终端样式按钮。适合开发者/OpenClaw 圈,信号强:"这是 hacker 工具,不是 chatGPT 又一个壳"。

Option C
Editorial · 杂志感

大衬线 + 报头 vol/issue 标识 + 两栏图文 + 厚轮廓按钮。最像"严肃产品",未来加 changelog/blog 顺手就有版面。
